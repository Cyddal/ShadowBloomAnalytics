# ShadowBloomAnalytics
Private competitive analytics platform for a Tier 3 women's League of Legends team.
ShadowBloom Analytics

ShadowBloom Analytics is an internal competitive analysis platform built for a Tier 3 women's League of Legends team that participates in structured tournaments and offline events with prize pools.

The project aims to support coaching, draft preparation, and player development by centralizing Riot API data, match history, and strategic insights into one unified platform.

Planned Features

Solo Queue performance tracker

Competitive match records database

Champion statistics dashboards

Draft pattern recognition tools

Searchable VoD library linked to match metadata

Objective timers and gold difference tracking

Vision heatmaps (ward placement density)

Jungle path visualizations

Jungle proximity analysis

Tech Stack

Frontend: React
Backend: Node.js (API layer)
Database: MongoDB Atlas
Hosting: Vercel

Project Status

Currently in early development.
Setting up infrastructure, database models, and Riot API integration.

Access & Scope

This platform is not intended for public use or monetization.
It will only be accessible to team players and staff for competitive improvement purposes.

Riot API Compliance

ShadowBloom Analytics will follow Riot Games API policies:

No resale or redistribution of data

Respect for rate limits

No exposure of private player data

Internal competitive usage only
